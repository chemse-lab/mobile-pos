# 🔄 دليل المزامنة (نموذج البائع: مشروع واحد)

<div dir="rtl">

في هذا النموذج **أنت** (البائع) تملك مشروع Supabase واحداً، وكل التجار يعملون عليه. كل تاجر معزول تلقائياً. التاجر **لا يفتح أي حساب ولا يدخل أي إعداد** — المزامنة تُفعَّل تلقائياً بكود التفعيل.

---

## الإعداد لمرة واحدة (أنت البائع)

### 1. أنشئ مشروع Supabase
- اذهب إلى [supabase.com](https://supabase.com) → New Project (منطقة Frankfurt).

### 2. أنشئ الجداول
- SQL Editor → الصق `supabase-schema.sql` → Run.

### 3. ضع مفاتيح المشروع في التطبيق
- من Settings → API انسخ **Project URL** و **anon public key**.
- افتح `index.html` وابحث عن `CONFIG` وضع القيمتين:

```javascript
const CONFIG={
  ...
  SUPABASE_URL:'https://xxxxx.supabase.co',   // ضع رابطك
  SUPABASE_ANON:'eyJhbGci...'                 // ضع مفتاحك
};
```

- ⚠ تأكد أن `SECRET` **نفسه** في `index.html` و `activation-tool.html`.

بعد هذا، ارفع التطبيق. **انتهى إعدادك.**

---

## كيف يعمل مع كل تاجر (تلقائي بالكامل)

```
1. التاجر يفتح التطبيق → 7 أيام تجربة
2. يرسل لك Install ID عبر WhatsApp
3. أنت تولّد كود التفعيل من activation-tool.html
4. التاجر يُدخل الكود → التطبيق يُفعَّل + المزامنة تُضبط تلقائياً ✓
```

**معرّف المتجر والرمز السري يُشتقّان تلقائياً من كود التفعيل** — لا يدخلهما التاجر ولا يراهما. كل تاجر يحصل على متجر فريد ومعزول.

---

## أجهزة متعددة لنفس التاجر (كاشير + هاتف)

بسيط جداً:
1. الجهاز الأول: التاجر يُفعّل بالكود العادي.
2. الجهاز الثاني: يُدخل **نفس الكود**، ويؤشّر على خانة **"هذا جهاز ثانٍ"**.
3. الجهازان الآن على نفس المتجر — البيانات تتزامن لحظياً.

---

## الأمان

- مفتاح Supabase عام (anon) — الأمان من RLS الذي يمنع أي وصول بدون الرمز السري الصحيح.
- الرمز السري لكل متجر مشتقّ من سرّك (HMAC) — لا يُخزَّن في الكود، يُولَّد فقط من كود التفعيل الصحيح.
- حتى لو رأى أحد المفتاح، لا يصل لبيانات أي متجر بدون كود تفعيل صالح.

---

## بديل: كل تاجر بمشروعه الخاص

إن فضّلت ألا تدير مشروعاً مشتركاً، اترك `SUPABASE_URL` و `SUPABASE_ANON` فارغين. عندها يظهر للتاجر زر "ربط المزامنة" اليدوي في الإعدادات، ويُدخل بياناته بنفسه (راجع الطريقة اليدوية).

## الطبقة المجانية

500MB + أجهزة غير محدودة + تزامن لحظي — تكفي عشرات المتاجر. للتوسّع الكبير، ترقية Pro (~25$/شهر).
