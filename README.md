# 📱 Mobile POS — نقطة بيع لمحلات الهواتف

<div dir="rtl">

**نظام إدارة متكامل لمحلات الهواتف والإكسسوارات — يعمل بدون إنترنت، بملف HTML واحد.**

مصمّم خصيصاً للسوق الجزائري: بالدينار الجزائري (دج)، بواجهة عربية RTL كاملة، مع دعم الفرنسية والإنجليزية.

</div>

## ✨ الميزات / Features

<div dir="rtl">

### 📊 لوحة تحكم ذكية
- مبيعات اليوم مع مقارنة بالأمس (نسبة مئوية)
- ربح اليوم، حالة المخزون، صيانات معلّقة، عدد العملاء
- رسم بياني للمبيعات 14 يوم
- توزيع طرق الدفع (نقد / بطاقة / تحويل / دين)
- تنبيهات المخزون المنخفض والصيانات الجاهزة والضمانات المنتهية

### 🛒 نقطة بيع (POS) احترافية
- شبكة منتجات + هواتف IMEI مع بحث فوري
- عربة تسوّق مع خصم، 4 طرق دفع
- الدفع بالدين → يُضاف تلقائياً على مديونية العميل
- وصل حراري 74mm جاهز للطباعة على أي طابعة عادية

### 📦 إدارة كاملة
- **المنتجات** — إكسسوارات، شواحن، كابلات، سماعات، حماية...
- **الهواتف IMEI** — تسجيل IMEI1/2، ذاكرة، RAM، بطارية، ضمان بالأشهر
- **الصيانة** — تذاكر مع دورة حياة (فحص → إصلاح → جاهز → تسليم)
- **العملاء** — سجل مشتريات + تتبع الديون + سداد
- **الموردون، المشتريات، المصروفات، الموظفون**

### 📈 تقارير مالية دقيقة
- إيراد، تكلفة، ربح إجمالي، ربح الصيانات، مصروفات، **ربح صافي**
- رسوم بيانية يومية
- أفضل المنتجات مبيعاً
- توزيع طرق الدفع
- تصدير Excel/CSV، طباعة

### 🔒 الأمان والحماية
- قفل PIN بأربعة أرقام
- نسخ احتياطية JSON كاملة
- استعادة من نسخة احتياطية
- تجربة مجانية 7 أيام
- تفعيل بكود آمن (HMAC-SHA256)

### 🌍 متعدد اللغات
- العربية 🇩🇿 (RTL)
- Français 🇫🇷
- English 🇬🇧

### 💾 يعمل بدون إنترنت 100%
- IndexedDB للتخزين المحلي
- بدون سيرفر، بدون Cloud
- بدون اشتراك شهري
- بدون npm، بدون build

## 🚀 التثبيت / Installation

<div dir="rtl">

### الطريقة 1: تصفح مباشر (الأسهل)
1. حمّل ملف `index.html`
2. افتحه بأي متصفح (Chrome, Firefox, Edge, Safari)
3. جاهز!

### الطريقة 2: GitHub Pages
- انشر الريبو على GitHub Pages → سيصبح لك رابط رسمي يعمل من أي جهاز.

### الطريقة 3: تطبيق أندرويد/iOS (Capacitor)
```bash
npm install -g @capacitor/cli
npx cap init "Mobile POS" com.yourbrand.mobilepos
npx cap add android
npx cap sync
# افتح في Android Studio وابنِ APK/AAB
```

### الطريقة 4: تطبيق سطح المكتب (Electron)
```bash
npm install -g electron
electron index.html
```

</div>

## 🎯 نظام التجربة والتفعيل / Trial & Activation

<div dir="rtl">

### للمستخدم النهائي:
1. **7 أيام تجربة كاملة** — كل الميزات، بدون قيود.
2. بعد انتهاء التجربة، تظهر شاشة قفل تطلب كود التفعيل.
3. النسخ الاحتياطية تبقى متاحة حتى بعد انتهاء التجربة.
4. المستخدم ينسخ **معرّف التثبيت** (Install ID) ويُرسله عبر WhatsApp.
5. بعد الدفع، يستلم كود التفعيل ويُدخله → تُفتح النسخة للأبد.

### للبائع:
- استخدم `activation-tool.html` (⚠ **لا ترفعه على GitHub علناً**) لتوليد الأكواد.
- الأداة تأخذ Install ID → تُنتج كود التفعيل الفريد لكل عميل.
- الأكواد لا تعمل إلا على الجهاز الذي طُلبت له.

## 🖨️ الطباعة / Printing

<div dir="rtl">

الوصل مصمّم لطابعات حرارية 74mm (مثل Xprinter, Epson TM series).  
يعمل أيضاً على طابعات A4 عادية (سيُصغَّر أوتوماتيكياً).

**اختصار سريع:** أثناء عرض الوصل، اضغط `Ctrl+P` لطباعته.

## 🛠️ التقنيات / Tech Stack

- **HTML5** + **CSS3** + **Vanilla JavaScript** (بدون فريمورك)
